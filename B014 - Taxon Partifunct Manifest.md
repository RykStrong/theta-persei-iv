# B014 – The Taxon Partifunct Manifest (TPM) and Modular Taxonomy Standards

## Summary

This dispatch introduces the **Taxon Partifunct Manifest (TPM)** as the standard framework for describing, comparing, and modelling composite organisms on TPIV. It defines how we use the term *taxon* in a planetary context where organisms are not genetically singular, but modular assemblies composed of distinct and independently evolved subsystems.

---

## I. Definition of "Taxon" on TPIV

In the PRXEMA/Green Horizon TPIV classification system, a **taxon** refers to a statistically coherent cluster of composite organisms that:
- Occupy a consistent ecological role or trophic niche
- Share a definable partifunct core set
- Present a bounded range of morphological divergence (MDI)
- Maintain a Partifunct Set Variance (PSV) of ≥60% within sample size n ≥ 50

Thus, a taxon on TPIV is not defined by lineage or single-genome inheritance, but by functional composition, reproductive architecture, and ecological behaviour.

---

## II. The Taxon Partifunct Manifest (TPM)

The TPM is a structured biological record containing:
1. Taxon Identifier (Name and shorthand code)
2. Ecological Description
3. Visual Description
4. Partifunct Core Set
5. PSV and MDI Values
6. Known Variants
7. Subsystem Interactions
8. GFP Sources
9. Reproduction Type (Womb/Non-Womb)
10. Observational Notes and Unresolveds

---

## III. TPM Format – Worked Example: Zebra Grazer

**Taxon Name:** Zebra Grazer
**Designation Code:** ZG-LWG-Hx-S *(Zebra Grazer / Large Walking Grazer / Hexapedal / Steppe)*
**Ecological Role:** Midweight plains grazer, steppe biome
**Visual Summary:** Hexapedal, striped dorsal surface, vertical sensory stalk, average adult mass ~500 kg, height 2.2 m + 1 m sensory stalk

### Partifunct Core Set

We are currently tracking the 12–15 most common partifuncts comprising approximately 90% of organismal mass and metabolic function. Minor partifuncts (<10% of total biomass) are not yet catalogued individually.

## IV. TPM in Practice

TPMs serve as the baseline records for taxonomic documentation. A TPM may include up to 30–50 Partifunct references, though only the core set (≥75% total body mass or ≥5% functional load) is required for standard registration.
