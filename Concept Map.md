# Concept Map: How TPIV Biology Fits Together

**A hierarchical guide to understanding the relationships between core concepts**

This page maps the logical dependencies and relationships between TPIV biological concepts. Follow the arrows to understand how ideas build on each other.

---

## The Big Picture

```
TPIV PLANET (Earth-like environment)
    ↓
NON-HOMOCHIRAL CHEMISTRY (Mixed chirality)
    ↓
MODULAR BIOLOGY (Incompatible systems must cooperate, not fuse)
    ↓
THREE LEVELS OF BIOLOGICAL ORGANIZATION
    ├─ GFP (Information molecules)
    ├─ Partifuncts (Functional modules)
    └─ Composite Organisms (Assembled creatures)
         ↓
    DISTRIBUTED ORGANISMS (Ultimate modularity)
```

## Level 1: Chemical Foundation

### The Starting Point
```
THETA PERSEI IV
├─ F8V star (higher UV than Sol)
├─ Earth-like gravity (1.166g)
├─ Earth-like atmosphere (1.2 atm, 25% O₂)
└─ Three moons (strong tidal forces)
```
**Pages:** [[Planet-Overview]], [[Atmosphere]], [[Moons-and-Tides]]

### Why Life Is Different
```
NON-HOMOCHIRAL CHEMISTRY
├─ Uses BOTH left & right-handed molecules
├─ Creates biochemical incompatibility barriers
└─ Forces modular evolution (cooperation not fusion)
    ↓
GENOMIC FOLDED PROTEINS (GFP)
├─ Replaces DNA as information storage
├─ Protein-based genetic code
└─ Bundles mix and match freely
```
**Pages:** [[Non-Homochirality]], [[GFP-and-Genodules]]

**Key Insight:** Mixed chirality prevents direct genetic fusion, driving evolution of cooperative modular systems.

## Level 2: Fundamental Units

### Information Flow
```
GFP (Genomic Folded Protein)
    ↓
MGFP Transcription Pathway
├─ MGFP-R (Read)
├─ MGFP-C (Copy)
├─ MGFP-G (Gather)
└─ MGFP-A (Assemble)
    ↓
PROTEINS (Function)
```
**Pages:** [[GFP-and-Genodules]], [[MGFP-Transcription-Pathway]]

### Structural Units
```
GFP → EXPRESS AS → BLU (Basic Life Units)
                       ├─ 100x larger than Earth cells
                       ├─ Don't contain full genome
                       └─ Form continuous tissue
```
**Pages:** [[21-BLUs-Basic-Life-Units]]

### Functional Organization
```
GFP BUNDLES (Genodules)
    ↓
EXPRESS AS
    ↓
PARTIFUNCTS
├─ Genetically distinct modules
├─ Specialized functions (limb, digestion, brain, etc.)
├─ Can express in multiple contexts
│   └─ Same GFP → grass stem OR animal leg OR wing
└─ Constantly shed genetic material
```
**Pages:** [[Partifunct Core Definition]], [[Multi-Expression-Potential]]

**Critical Distinction:**
```
EARTH                          TPIV
Organ = shares DNA    ≠    Partifunct = unique DNA
Zooid = shares DNA    ≠    Partifunct = unique DNA
```
**Page:** [[Partifunct-vs-Organ-vs-Zooid]]

## Level 3: Organism Assembly

### How Organisms Form
```
MULTIPLE PARTIFUNCTS
    ↓
INTEGRATE VIA
├─ Vascular interfaces (Transfer Organs)
├─ Neural connections (Central Nerve Loop)
├─ Hormonal signaling
└─ Metabolic coordination
    ↓
COMPOSITE ORGANISM
├─ Fully integrated functional animal
├─ Each part genetically distinct
└─ Appears as single individual
```
**Pages:** [[Composite-Organisms]], [[24a-Integration-Mechanisms]]

### Result: Non-Homogeneity
```
SAME TAXON, DIFFERENT INDIVIDUALS
├─ Share core partifunct types (PSV = similarity)
├─ Vary in expression (MDI = diversity)
└─ No two individuals genetically identical
    
EXAMPLE: Zebra Grazer Herd
├─ Some have 6 legs, some 8
├─ Some have 1 mouth, some 2
├─ Some have elaborate sensors, some minimal
└─ All recognizably "Zebra Grazer"
```
**Pages:** [[24b-Non-Homogeneity]], [[31-Taxa-and-Metrics-PSV-MDI]]

## Level 4: Reproduction & Genetics

### Environmental Genetics
```
ADULT ORGANISMS
    ↓
SHED GFP INTO ENVIRONMENT
    ↓
eGFP (Environmental GFP)
├─ Carried by mites
├─ Carried by pollen-analogs
├─ Carried by SBLUs (Symbiotic BLUs)
└─ Floats in air, water, food
    ↓
COLLECTED BY OTHER ORGANISMS
```
**Pages:** [[Environmental-GFP]], [[GFP-Carriers-and-Vectors]]

### Womb-Based Reproduction
```
ADULT COMPOSITE ORGANISM
    ↓
Each Partifunct:
├─ Collects compatible eGFP
├─ Recombines with own GFP
└─ Creates PARTIFZYGOTE
    ↓
WOMB PARTIFUNCT receives partifzygotes
├─ Evaluates compatibility
├─ "Polls" parent partifuncts
├─ Assembles viable combination
└─ Achieves metabolic integrity
    ↓
BIRTH (new composite organism)
├─ May differ from parent
├─ Mosaic of familiar + novel parts
└─ Member of same taxon (usually)
```
**Pages:** [[Womb-Enabled-Reproduction]], [[3B-Womb-Partifunct-Systems]]

### Non-Womb Reproduction
```
SIMPLE ORGANISMS ("Plants")
    ↓
Shed BLU-wrapped GFP packets
    ↓
Environmental conditions trigger:
├─ Self-organization
├─ Spontaneous assembly
└─ Metabolic activation
    ↓
FREE-ASSEMBLING ORGANISM
```
**Pages:** [[Non-Womb-Reproduction]]

## Level 5: Evolution & Selection

### Multi-Level Selection
```
NATURAL SELECTION acts on:

1. GFP LEVEL
   └─ Which genetic packets survive/spread?
   
2. PARTIFUNCT LEVEL
   └─ Which modules integrate successfully?
   
3. COMPOSITE LEVEL
   └─ Which assemblies thrive?
   
4. TAXON LEVEL
   └─ Which configurations persist?
```
**Pages:** [[Natural-Selection-on-TPIV]]

### Evolutionary Strategies
```
PARTIFUNCT FITNESS DEPENDS ON:
├─ Versatility (works with many other partifuncts)
├─ Environmental spread (many expression contexts)
└─ Assembly success (selected by wombs)

COMPOSITE FITNESS DEPENDS ON:
├─ Viable partifunct combinations
├─ Metabolic integration
└─ Ecological niche success

RESULT: Tension between:
├─ CONSERVATISM (replicate proven combinations)
└─ INNOVATION (try new assemblies)
```
**Pages:** [[Reproductive-Strategies]]

EARTH TAXONOMY          TPIV TAXONOMY
├─ Species (genetic)  →  Taxa (statistical)
├─ Linnean hierarchy  →  Functional classification

## Level 7: Advanced Organization

### Standard Composite Organisms
```
TYPICAL PATTERN (e.g., Zebra Grazer)
├─ Hexagirdle body plan
├─ 6-8 limbs
├─ Central nervous loop
├─ Distributed hearts & lungs
├─ Womb partifunct
└─ Sensory stalk
```
**Pages:** [[3A-Hexagirdle-Octagirdle]], [[TPM-Zebra-Grazer]]

### Separated Behaviors (SB)
```
COMPOSITE ORGANISM
    ↓
Temporarily detaches EXOVANT
├─ Throwjaw (predatory strike)
├─ Tossclaw (manipulation)
└─ Drophunter hunters (aerial predation)
    ↓
Exovant performs task
    ↓
Reintegrates with parent
```
**Pages:** [[56-Separated-Behaviors-and-Exovants]]

**Key:** Exovant is NOT viable independently; must return to parent.

### Joined Behaviors (JB)
```
MULTIPLE INDEPENDENT ORGANISMS
    ↓
Temporarily join physically
├─ Stringbirds (chain for flight)
└─ Ribbonfish (stack for filter-feeding)
    ↓
Perform task cooperatively
    ↓
Separate (each viable independently)
```
**Pages:** [[55-Combinant-Behaviors]]

**Key:** Each organism is independently viable; joining is temporary strategy.

### Distributed Organisms (DO)
```
SINGLE ORGANISM
    ↓
Exists as MULTIPLE PARTIFVANTS
├─ Physically separated bodies
├─ Each specialized (feeding, digesting, sensing, etc.)
├─ None viable independently
└─ Communicate via CEPs
    ↓
DO-MITES carry CEPs
├─ Insect-like carriers (Apanon)
└─ OR Radio waves (Kolofluke)
    ↓
SYNCHRONIZED BEHAVIOR
└─ One reproductive entity
```
**Pages:** [[50-Distributed-Organisms-Overview]], [[51-CEP-and-Do-Mites]]

**Critical Distinction:**
```
TYPE          VIABLE SEPARATELY?    COMMUNICATION
SB Exovant    NO (must rejoin)      Direct (still attached)
JB Members    YES (independent)      Passive/chemical
DO Partifvants NO (permanent)       CEPs via do-mites
```
**Page:** [[50a-DO-vs-SB-vs-JB-Comparison]]

## Key Relationships

### Causality Chains

**Why modularity exists:**
```
Non-homochiral chemistry 
    → Biochemical incompatibility 
    → Must cooperate, can't fuse 
    → Modular evolution
```

**Why no species:**
```
Horizontal gene transfer (eGFP) 
    → Offspring ≠ parents genetically 
    → Only statistical clustering possible 
    → Taxa, not species
```

**Why high diversity within taxa:**
```
Environmental GFP varies 
    → Womb assembles available partifzygotes 
    → Each offspring different combination 
    → High MDI
```

**Why DOs evolved:**
```
Island isolation (Apanon, Kolofluke) 
    → Limited partifunct diversity 
    → Pressure to maximize efficiency 
    → Distribute functions across bodies
```

### Bidirectional Relationships

```
Partifuncts ←→ Composite Organisms
├─ Partifuncts need composites (can't survive alone)
└─ Composites need partifuncts (can't form without them)

eGFP dispersal ←→ Organism success
├─ Successful organisms shed more eGFP
└─ Abundant eGFP enables more offspring

Culture ←→ Genetics
├─ Cultural behaviors affect which partifuncts spread
└─ Partifuncts (esp. brain types) enable culture
```

## Concept Difficulty Map

### Difficulty Level 1 (Easy)
**Concepts that map easily to Earth biology:**
- Planet is Earth-like
- Organisms have limbs, mouths, eyes
- Predator-prey relationships
- Herding behavior
- Death and decomposition

### Difficulty Level 2 (Moderate)
**Requires adjusting Earth assumptions:**
- No DNA, but GFP serves similar role
- No cells, but BLUs serve similar role
- No species, but taxa serve similar role
- Convergent evolution produces familiar forms

### Difficulty Level 3 (Challenging)
**Fundamentally different from Earth:**
- Each body part genetically distinct
- Offspring don't resemble parents genetically
- Genetic material floats in environment
- No two individuals genetically identical
- "Plant" and "animal" don't apply

### Difficulty Level 4 (Mind-Bending)
**Challenges core assumptions about biology:**
- One organism in multiple separated bodies (DO)
- Communication via insect carriers
- Brain distributed across genetic modules
- Organisms farming their own genes
- Radio-wave communication (Kolofluke)

## Common Misconceptions

### ❌ Wrong → ✅ Right

**"Partifuncts are organs"**
- ❌ Organs share DNA with whole body
- ✅ Partifuncts have unique DNA, genetically distinct

**"Taxa are species"**
- ❌ Species have fixed genetic boundaries
- ✅ Taxa are statistical clusters, fluid boundaries

**"Offspring inherit parents' genes"**
- ❌ Not direct inheritance
- ✅ Partifuncts collect eGFP from environment, recombine

**"Distributed Organisms are colonies"**
- ❌ Colonies = many genetic individuals
- ✅ DOs = one biological individual in many bodies

**"Womb is like mammal pregnancy"**
- ❌ Mammal womb grows one genome
- ✅ TPIV womb assembles multiple genomes

**"Plants and animals are distinct"**
- ❌ No fundamental biochemical difference
- ✅ Continuum based on energy source and locomotion

## Learning Pathways Through Concept Space

### Path A: Bottom-Up (Molecular → Organism)
```
1. Chemistry (GFP, BLUs)
2. Modules (Partifuncts)
3. Assembly (Composite organisms)
4. Classification (Taxa)
5. Advanced (DOs, behaviors)
```
**Best for:** Scientists, systematic learners

### Path B: Top-Down (Organism → Molecular)
```
1. See organism (Zebra Grazer)
2. Understand it's modular (Partifuncts)
3. Learn how modules form (GFP, BLUs)
4. Understand reproduction (Assembly-conception)
5. Explore diversity (Taxa, DOs)
```
**Best for:** Visual learners, naturalists

### Path C: Comparative (TPIV vs Earth)
```
1. Start with similarities (planet, ecology)
2. Identify differences (GFP not DNA, etc.)
3. Understand why different (non-homochirality)
4. Explore implications (modularity, taxa)
5. Discover unique features (DOs, radio organisms)
```
**Best for:** Earth biologists, comparative thinkers

### Path D: Problem-Based (Answer questions)
```
Start with any question:
"How does X work?" → Follow concept links
"Why is Y different?" → Trace causality chains
"What is Z?" → Build from dependencies
```
**Best for:** Self-directed learners, researchers

## Integration Points

### Where Concepts Meet

**GFP + Environment = eGFP system**
- Enables rapid evolution
- Creates genetic marketplace
- Drives non-homogeneity

**Partifuncts + Womb = Assembly-conception**
- Replaces sexual reproduction
- Allows quality control
- Maintains taxon stability

**Modularity + Island isolation = Distributed Organisms**
- Maximum resource efficiency
- Novel communication systems
- Blur individual/collective boundary

**Brain partifuncts + Behavior = Culture**
- Learned behaviors affect reproduction
- Genetic-cultural co-evolution
- Tool use and engineering

**Multi-expression + Selection = Ecological versatility**
- Same partifunct in multiple niches
- Rapid niche filling
- Convergent yet alien solutions
