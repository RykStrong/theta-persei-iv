# DO-HillWolf PackHunter

**Classification:** Stalking/Opportunistic Pack Hunter (Distributed)  
**Biome:** Hill country, various continents  
**PSV:** 68% (±1σ)  
**MDI:** 6.2  
**Communication:** Chemical [do-mites](CEP%20and%20Do-Mites.md)

---

## Overview

### Chemical-Based Predator DO

**DO-HillWolf PackHunter** represents an apex predator strategy using [distributed organism](Distributed%20Organisms%20Overview.md) architecture with chemical [do-mite](CEP%20and%20Do-Mites.md) communication.

**Key Features:**
- Multiple specialized [partivant](Partivant%20Definition.md) types (scout, attack, feeding, cognitive)
- Do-mite cloud coordination
- Coordinated pack hunting distributed across individual bodies
- Wide-area resource exploitation

---

## Partivant Types

### Specialized Hunting Bodies

**Scout Nodes:**

**Structure:**
- Lean, agile build
- Enhanced [sensory organs](Sensory%20Systems.md) (vision, hearing, smell)
- Long legs for speed and endurance
- Minimal digestive system

**Function:**
- Locate prey at distance
- Track movement patterns
- Relay position data via [do-mites](CEP%20and%20Do-Mites.md)
- Monitor terrain and escape routes

**Behavior:**
- Spread across hunting area (100-500m typical)
- Constant do-mite communication with attack nodes
- Visual/auditory prey detection
- Pheromone trail marking

---

### Attack Nodes

**Structure:**
- Heavy muscular build
- Powerful jaws (killing bite)
- Enhanced [MOTO brain](Nervous%20Systems.md) (coordination)
- Claws for grappling
- Medium digestive capacity

**Function:**
- Execute coordinated ambush
- Kill large prey
- Coordinate with other attack nodes via do-mite signals
- Initial feeding (high-energy organs)

**Behavior:**
- Receive position data from scouts (do-mite carriers)
- Coordinate approach vectors
- Multi-point attack when possible
- Pack tactics distributed across bodies

**Coordination Example:**
- Scout node detects prey 200m away
- [Do-mites](CEP%20and%20Do-Mites.md) carry position to attack nodes
- Attack nodes converge from different directions
- Coordinated assault (chemical signal synchronized)
- Prey escape routes blocked by scouts

---

### Feeding Nodes

**Structure:**
- Large body cavity
- Extensive digestive system
- Strong jaws (bone-crushing)
- Reduced mobility (heavy build)
- Storage capacity (fat reserves)

**Function:**
- Process kills made by attack nodes
- Break down entire carcass
- Extract maximum nutrition
- Store energy for lean periods
- Distribute nutrients to other partivants via do-mites

**Behavior:**
- Move to kill site after attack complete
- Thorough carcass processing
- May remain at site for extended periods
- Other partivants continue hunting

---

### Food Processing Nodes

**Structure:**
- Massive fermentation chambers
- Stationary or very slow-moving
- Symbiotic gut microbes
- Chemical processing capability

**Function:**
- Process tougher tissues (bones, hide, cartilage)
- Extract nutrients attack nodes can't digest
- Long-term nutrient storage
- Supplement diet during prey scarcity

**Distribution:**
- Positioned in safe den sites
- Hidden in cave systems or dense brush
- Attack/feeding nodes bring food periodically
- Most vulnerable partivant type (rarely moves)

---

### Sensory Nodes

**Structure:**
- Large ear-like structures
- Enhanced [visual organs](Sensory%20Systems.md)
- Chemical sensors (olfactory)
- Elevated position capability (long neck or climbing)

**Function:**
- Long-range threat detection
- Monitor territory boundaries
- Track prey herds over time
- Coordinate with scout nodes

**Specialization:**
- Enhanced hearing (vibration detection)
- Chemical sensing (pheromone tracking)
- Visual monitoring (elevated positions)
- [Sensory integration](Distributed%20Organisms%20Overview.md) distributed

---

### Cognitive Nodes

**Structure:**
- Multiple [brain partifuncts](Nervous%20Systems.md) (MEMO, STRA, SOCI, NAVI)
- Well-protected body position
- Moderate size and mobility
- Enhanced do-mite reception/production

**Function:**
- [Distributed cognition](Distributed%20Organisms%20Overview.md) coordination
- Strategic planning (hunt strategies)
- Memory storage (territory maps, prey patterns, successful tactics)
- Decision-making integration across all partivants
- Do-mite signal coordination (communication hub)

**Intelligence:**
- Complex cognitive capabilities
- Learn from failed hunts
- Adapt tactics to prey behavior
- Social coordination with other DO-HillWolf organisms

## Do-Mite Communication System

### Chemical Coordination

**[Swarm Characteristics](CEP%20and%20Do-Mites.md):**
- Thousands of do-mites per DO
- [Corridors](CEP%20and%20Do-Mites.md) connect partivants
- Dense envelope around cognitive nodes
- High traffic during hunts

**Signal Types:**
- **Position data:** Scout locations, prey locations (chemical markers)
- **Threat alerts:** Predator warnings, danger pheromones
- **Coordination:** Attack timing signals, movement hormones
- **Status updates:** Partivant health, hunger levels
- **Memory queries:** Chemical markers trigger stored patterns

**Range Limitation:**
- Effective range: 200-500m typical
- Maximum observed: ~1km (favorable conditions)
- Weather-dependent (wind disperses signals)
- Dense vegetation blocks corridors

## Hunting Strategy

### Coordinated Pack Behavior

**Phase 1 - Detection:**
- Scout partivants patrol territory
- Detect prey via vision/sound/smell
- Release marker pheromones
- [Do-mites](CEP%20and%20Do-Mites.md) carry signals to cognitive nodes
- Cognitive nodes evaluate opportunity

**Phase 2 - Positioning:**
- Attack nodes receive movement instructions via do-mites
- Approach from multiple vectors
- Use terrain for concealment
- Pheromone trails guide positioning
- Scout nodes provide updates

**Phase 3 - Ambush:**
- Cognitive node releases "attack" hormone cascade
- Do-mites distribute signal to all attack nodes
- Near-simultaneous multi-point attack
- Scout nodes block escape routes
- Attack nodes execute kill

**Phase 4 - Processing:**
- Attack nodes feed on high-energy organs first
- Do-mites signal feeding nodes
- Feeding nodes move to carcass
- Food processing nodes receive portions later
- Cognitive nodes store hunt success data

## Territory and Range

### Limited by Communication Range

**Territory Size:**
- 5-15 km² typical
- Limited by do-mite effective range
- Smaller than radio-based DOs
- Defended against other DO-HillWolf packs

**Partivant Distribution:**
- Scout partivants: Dispersed (200-500m apart)
- Attack partivants: Clustered in 2-3 groups
- Feeding partivants: Mobile but slower
- Processing partivants: Fixed den sites
- Cognitive partivants: Central position
- Maximum separation: ~1km (extreme cases)

**[Corridor Maintenance](CEP%20and%20Do-Mites.md):**
- Established flight paths between key partivants
- Pheromone trail reinforcement
- Higher do-mite density along corridors
- Break in corridor = communication disruption

## Reproduction

### Complex Assembly Challenge

**Reproductive Partivants:**
Created as needed, typically in denning season.

**Process:**
- One feeding or cognitive partivant develops [womb partifunct](Womb%20Partifunct%20Systems.md)
- Or dedicated reproductive partivant assembled
- Collects [eGFP](Environmental%20GFP.md) from prey and environment
- Assembles multiple offspring partivants
- Birth produces complete DO (all partivant types)
- Do-mite communication functional immediately

**Challenge:**
- Must create multiple specialized partivants
- Do-mite production and navigation must work from birth
- High failure rate—coordinating multiple partivants complex
- Non-viable sets die within days

**Development:**
- Initial birth: 4-8 partivants (core set)
- Additional partivants added over time
- Full adult complement: 12-20 partivants
- Maturation: 2-3 local years

## Ecological Role

### Apex Predator

**Prey:**
- Large herbivores (standard composites)
- Medium-sized grazers
- Other predators (opportunistic)
- Occasionally [DO-ScrubDeer](DO%20ScrubDeer%20Browser.md) (difficult due to their distribution)

**Population Control:**
- Regulates herbivore populations
- Prevents overgrazing
- Maintains ecosystem balance
- [Selection pressure](Natural%20Selection%20on%20TPIV.md) on prey tactics

**Competition:**
- Other predator DOs (territorial conflicts)
- Standard composite predators (usually outcompeted)
- [Do-mite airspace conflicts](DO-Mite%20conflict.md) with neighboring packs

---

## Vulnerabilities

### Communication Disruption

**Environmental:**
- High winds scatter do-mite swarms
- Rain grounds do-mites temporarily
- Dense vegetation blocks corridors
- Temperature extremes affect do-mite flight

**Biological:**
- Predators targeting do-mite clouds
- Other DOs disrupting swarms ([airspace conflicts](DO-Mite%20conflict.md))
- Parasites attacking do-mites
- Chemical jamming (pheromone confusion)

**Partivant Loss:**
- Cognitive node loss most critical (coordination breakdown)
- Scout loss reduces detection range
- Attack node loss reduces hunting success
- Processing node loss forces reliance on attack/feeding nodes only
- Can regenerate lost partivants over time

---

## Comparison to Standard Pack Hunters

### DO vs Non-DO Predators

**DO-HillWolf Advantages:**
- Extreme specialization of partivants
- Loss of individuals less catastrophic
- Wide-area sensory coverage
- Parallel processing (distributed cognition)

**Disadvantages:**
- Communication overhead (do-mite maintenance)
- Coordination complexity
- Vulnerable to communication disruption
- Higher assembly failure rate

**Standard Pack Hunters:**
- Each individual fully viable
- Simpler coordination
- Lower overhead
- But less specialized, more vulnerable to losses

## Behavioral Observations

### Pack Dynamics

**Coordination Complexity:**
- Scout nodes act as "eyes"
- Cognitive nodes act as "brain"
- Attack nodes act as "weapons"
- Feeding nodes act as "stomach"
- All connected by do-mite "nervous system"

**Learning:**
- Failed hunts stored in cognitive partivant memory
- Successful tactics reinforced
- Adapt to prey defensive behaviors
- [MEMO brain](Nervous%20Systems.md) accumulates experience

**Social Interactions:**
- Multiple DO-HillWolf organisms may cooperate temporarily
- Territorial defense coordinated
- Information sharing via pheromone markers
- Possible [combinant behaviors](Combinant%20Behaviors.md) (unconfirmed)

## Research Observations

### Population Density

**Moderate Density:**
- 2-4 packs per 100 km²
- Higher than [DO-ScrubDeer](DO%20ScrubDeer%20Browser.md) (predator vs prey)
- Territory size limited by do-mite range
- Seasonal variation (denning affects distribution)

**Pack Composition:**
- 12-20 partivants per pack typical
- Larger packs in prey-rich areas
- Smaller in marginal habitats
- Seasonal addition of reproductive partivants
