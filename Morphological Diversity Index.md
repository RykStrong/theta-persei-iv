# Morphological Divergence Index (MDI)

**Definition:** Numerical measure of how much morphological variation exists in actual expression (physical form) within a group sharing a partifunct set  
**Scale:** Typically 0-10+ (higher = more variation)  
**Low MDI:** Uniform appearance  
**High MDI:** Significant morphological diversity

---

## What MDI Measures

### Physical Variation

MDI is expressed numerically to indicate how much morphological variation exists in the actual expression (physical form) of a given set of partifuncts within the composite group.

**A lower MDI (<2)** means very uniform shape; **higher values (>5)** indicate significant morphological diversity.

---

## Example Statement

**Standard Reporting Format:**

*"MDI of Savannah Bison Grazer = 3.2."*

**Meaning:** Moderate variation; herd members display noticeable but generally minor morphological differences (e.g., some have extra sensory organs, variable limb count, etc.).

---

## MDI Ranges and Interpretation

### Low MDI (<2)

**Characteristics:**
- Very uniform appearance
- Minimal visible variation
- Tight morphological constraints
- Predictable phenotype

**Causes:**
- Narrow environmental conditions
- Strong selection for specific form
- Limited [expression plasticity](Multi%20Expression%20Potential.md)

---

### Moderate MDI (2-5)

**Characteristics:**
- Noticeable variation within recognizable pattern
- Some individuals larger, different proportions
- Variable optional features
- Still clearly same taxon

**Examples:**
- Zebra Grazer: MDI 2.8
- Crimson Striders: MDI 4.6
- Most established taxa fall here

---

### High MDI (>5)

**Characteristics:**
- Significant morphological diversity
- Same partifuncts express very differently
- Difficult to recognize as single group
- May appear as multiple taxa

**Examples:**
- LBB Feeders: MDI 8.2
- River-crossing populations: MDI 6.1
- Experimental or transitional groups

---

## What Affects MDI

### Expression Plasticity

**[Multi-Expression](Multi%20Expression%20Potential.md) Partifuncts:**
- Same genodule → many forms
- [MGFP-G](MGFP%20Transcription%20Pathway.md) interprets womb signals differently
- Environmental variation increases MDI
- ST-LMB can appear as thick leg OR thin leg

---

### Environmental Heterogeneity

**Variable Conditions:**
- Different temperatures → different bone density
- Different food availability → different sizes
- Different predator types → different defensive structures
- [Womb](Womb%20Enabled%20Reproduction.md) adapts each offspring

**Example:**
*"The population sampled near the Olan River crossing exhibited a PSV of 92% (indicating high partifunct consistency) but had an elevated MDI of 5.4. The river-crossing behavior appears to encourage wider morphological expression—approximately 35% more variable than average populations, likely reflecting adaptive pressures."*

---

### Optional Partifuncts

**Additional Modules:**
- Some individuals have extra sensory organs
- Some have defensive structures
- Some have enhanced mobility features
- Increases visible variation without changing core set

---

## MDI in Combined Analysis

### With PSV

**PEG Visualization:**
Scientists plot organisms on **Partifunct Expression Graph (PEG)**, where:
- Each dimension = trait or partifunct expression level
- MDI describes the **spread** of the point cloud
- [PSV](Partifunct%20Set%20Variance.md) describes the **clustering** consistency

**Result:**
- **Tight cluster (low MDI)**: Uniform appearance
- **Diffuse cloud (high MDI)**: Variable appearance

## Research Applications

### Field Identification

**Visual Assessment:**
- Measure size range
- Count limb variations
- Note optional features
- Calculate MDI from sample

**Challenges:**
- High MDI makes field ID difficult
- May require genetic confirmation
- Intermediate forms complicate assignment

---

### Evolutionary Monitoring

**MDI Changes Over Time:**
- **Rising MDI:** Population adapting to variable conditions
- **Falling MDI:** Selection narrowing optimal form
- **Stable MDI:** Equilibrium reached

---

### Adaptive Radiation

**High MDI Zones:**
- [Tidal zones](Littoral%20Zones.md): Variable environment → high MDI
- Marginal habitats: Exploration ongoing
- Recent invasions: Multiple strategies tested
