# Partifunct Set Variance (PSV)

**Definition:** Describes how consistently a given set of partifuncts occurs within the observed composite group  
**Measurement:** Percentage of individuals sharing the group's common core of partifuncts within ±1 standard deviation  
**High PSV:** Stable, consistent taxon  
**Low PSV:** Variable, transitional, or diffuse group

---

## What PSV Measures

### Partifunct Consistency

PSV describes how consistently a given set of partifuncts occurs within the observed composite group.

**Presented as:** A percentage indicating how many individual organisms share the group's common core of partifuncts within ±1 standard deviation.

---

## Example Statement

**Standard Reporting Format:**

*"PSV of Savannah Bison Grazer (LGW-Hx-2M) = 84.6% within 1σ."*

**Meaning:** 84.6% of observed individuals contain the standard partifunct set within one standard deviation of variance from the average.

---

## PSV Ranges and Interpretation

### High PSV (>75%)

**Characteristics:**
- Stable, consistent taxon
- Core partifunct set well-defined
- Recognizable body plan
- Predictable assemblies

**Examples:**
- Zebra Grazer: PSV 83%
- Most successful, established taxa

---

### Moderate PSV (50-75%)

**Characteristics:**
- Some variation in partifunct set
- Multiple viable configurations
- Transitional or adaptive populations
- Regional variants

**Examples:**
- River-crossing populations: PSV 76%
- Marginal habitat specialists

---

### Low PSV (<50%)

**Characteristics:**
- High partifunct variability
- Diffuse assemblages
- Catch-all categories
- May represent multiple emerging taxa

**Examples:**
- LBB Feeders: PSV 23%
- Highly experimental assemblages

---

## What Affects PSV

### Environmental Stability

**Stable Environments:**
- High PSV (proven combinations replicate)
- [Wombs](Womb%20Enabled%20Reproduction.md) favor conservative assembly
- Selection reinforces successful pattern

**Variable Environments:**
- Lower PSV (multiple strategies viable)
- Wombs experiment more
- Adaptive pressure for flexibility

---

### eGFP Availability

**Abundant [eGFP](Environmental%20GFP.md):**
- Wombs have many options
- More experimentation
- May lower PSV

**Limited eGFP:**
- Wombs use available material
- Less choice = more variation
- Or higher PSV if only one configuration works

---

### Population History

**Established Taxa:**
- Long history = stable partifunct sets
- High PSV common
- Self-reinforcing pattern

**Emerging Taxa:**
- New assemblages = variable sets
- Low PSV initially
- May stabilize or fragment

---

## PSV in Research

### Field Use

**Quick Assessment:**
- Sample population
- Identify core partifunct set (appears in >50%)
- Calculate percentage sharing that set
- Assign PSV value

**Genetic Confirmation:**
- Tissue sampling from multiple individuals
- [GFP sequencing](GFP%20Structure.md) from each partifunct
- Phylogenetic analysis
- Statistical clustering

---

### Monitoring Over Time

**PSV Changes Indicate:**
- Environmental shifts (PSV drops)
- Population stabilization (PSV rises)
- Speciation events (PSV split into two groups)
- Taxon collapse (PSV drops to near zero)

---

## PSV + MDI Combined

### Four Quadrant Model

**High PSV + Low [MDI](Morphological%20Diversity%20Index.md):**
- Stable, uniform taxon
- Example: Standard Zebra Grazer (PSV 83%, MDI 2.8)

**High PSV + High MDI:**
- Stable but variable taxon
- Same partifuncts, different expressions
- Example: Crimson Striders (PSV 78%, MDI 4.6)

**Low PSV + Low MDI:**
- Uniform appearance, variable genetics
- Convergent assembly from different partifuncts
- Rare pattern

**Low PSV + High MDI:**
- Chaotic diversity
- Example: LBB Feeders (PSV 23%, MDI 8.2)
